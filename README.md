## ⚡ Quick start

```bash
cd xlri-section-wars-dark   # or xlri-section-wars-fair
npm install
npm run dev                  # → localhost:5173
```

---

## 🧩 What's inside (both editions)

| 🎯 | Feature |
|---|---|
| 🎥 | Cinematic multi-scene loader (~4–6s) |
| 🖱️ | Custom cursor + mouse parallax hero |
| 🔥 | Canvas particles (embers 🔴 / confetti 🎊) |
| 🎴 | Square team-photo cards with hover glow |
| 🎵 | Looping background score, mute toggle |
| 📜 | Scroll-reveal animations, zero libraries |

Pure **React + plain CSS**. No Tailwind, no Bootstrap, no animation packages — just clean, commented, interview-ready code.
